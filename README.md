Research Collaboration Platform
Overview

The Research Collaboration Platform is a web-based application designed to connect researchers, students, and academic teams. It provides a centralized space to share research work, collaborate on projects, and manage academic interactions efficiently.

The platform aims to simplify collaboration by offering structured access to research teams, project opportunities, and researcher profiles.

Features
User authentication and profile management
CV upload and analysis
Research team exploration
Project and collaboration opportunities
Interactive and responsive user interface
Backend API for handling data and requests
Technologies Used

CSS

Backend
Python (Flask)

Installation and Setup
1. Clone the repository
git clone https://github.com/Nada-belmiloud/research_collab_platform.git
cd research_collab_platform
2. Backend setup
cd backend
pip install -r requirements.txt
python app.py
3. Frontend setup

Open the frontend folder and run the project using a live server or open the HTML file in your browser.
